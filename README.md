# VCF 9 Home Lab

This repository documents my VMware Cloud Foundation (VCF) 9 home lab build, validation process, and architecture references.

---

## Overview

This lab is designed to simulate real-world VCF deployment scenarios, including:

- Cloud Builder bring-up
- Offline depot configuration
- DNS and NTP validation
- ESXi host preparation
- vSAN and NSX design considerations
- Management Domain deployment
- Workload Domain expansion
- Troubleshooting and validation workflows

---

## Purpose

The goal of this repository is to serve as:

- A technical reference for VCF architecture and deployment
- A structured knowledge base for troubleshooting VCF environments
- A foundation for customer-facing designs and implementation practices

---

## Key Areas Covered

### Infrastructure & Bring-Up
- Cloud Builder deployment workflow
- VCF bring-up validation checks
- Offline depot setup and troubleshooting

### Networking
- NSX architecture (T0 / T1 / Edge)
- VLAN and overlay design
- MTU and connectivity validation

### Compute & Storage
- ESXi host configuration
- vSAN ESA considerations
- Disk validation and HCL awareness

### Operations & Lifecycle
- SDDC Manager workflows
- LCM / bundle validation
- Common failure scenarios and resolutions
